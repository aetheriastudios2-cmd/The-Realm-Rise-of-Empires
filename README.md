# The Realm: Rise of Empires

An open-world fantasy RPG built in Unreal Engine 5 by Aetheria Studios.

## Core Features

- Kingdom Building
- Dragon Riding
- Airship Fleets
- Mage Academy
- Dynamic Diplomacy
- Dark Lord Path
- Faction Warfare
- Massive Open World

## Play As

- Knight
- Lord
- King
- Wizard
- Dragon Rider
- Airship Captain
- Merchant
- Mercenary
- Dark Lord
